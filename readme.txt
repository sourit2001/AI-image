# AI 图像编辑专业网页需求说明

## 项目目标
打造一个专业、干净、用户友好的网页，结合 Replicate API（以 flux-kontext 及相关模型为主），为用户提供高质量的 AI 图像编辑服务。

## 核心功能
1. **背景消除**：上传图片后，支持一键去除背景，输出透明 PNG。
2. **添加背景生成**：为抠图后的主体自动生成、替换或上传自定义背景，并合成新图片。
3. **高清放大**：通过 AI 超分辨率模型，将图片无损放大，提升清晰度。
4. **一键消除文字**：自动检测并去除图片中的文字或水印，保持图片完整性。
5. **订阅功能**：集成 Stripe 支付，支持免费试用和高级会员订阅，订阅后解锁全部功能和更高额度。
6. **用户认证**：支持 Google 邮箱一键登录，或普通邮箱注册/登录。

## 技术栈建议
- 前端：Next.js（React）、Tailwind CSS（极简/专业风格）
- 后端：Next.js API routes
- AI 服务：Replicate API（flux-kontext、remove-bg、Real-ESRGAN、inpainting等）
- 订阅/支付：Stripe
- 用户认证：NextAuth.js（Google OAuth + Email magic link）
- 部署：Vercel

## 页面与交互
- 首页：上传图片，选择功能，实时预览，操作历史，对比原图与处理结果
- 订阅页：套餐、价格、权益展示，Stripe 支付入口
- 登录/注册页：Google 登录/邮箱注册
- UI风格：页面极简、专业，大量留白，按钮简洁直观，操作流程流畅，用户体验友好

## 用户流程示例
1. 用户登录/注册
2. 上传图片
3. 选择所需功能（如“背景消除”）
4. 预览处理效果，下载或继续编辑
5. 超出免费额度或需高级功能时，提示订阅
6. 订阅后解锁所有功能
